## 1. 同步文档

执行 sync-docs 的步骤: 更新任务文件 related, 更新相关 doc, 检查 AGENTS.md, commit + push.

## 2. 生成交接摘要

输出一段可直接粘贴给新 session 的文本, 包含:

### 当前状态
- 各页面完成度 (已完成 / 核心完成 / 未开始)
- 剩余 Node/Electron 依赖清单

### 下一步
- 具体可执行的动作 (文件 + 步骤描述)
- 优先级排序

### 避坑提醒
- 本阶段遇到的关键问题和解决方案
- 已验证无效的方案 (避免重复尝试)

### 开放问题
- 待用户确认或决策的项

### 关键 Commit
- 最近 3-5 个有意义的 commit hash + 说明
